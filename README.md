# ProductPhotoCapture

Phase 1: Minimal full-stack scaffold.

## Stack
- Server: Express, better-sqlite3, CORS, dotenv
- Client: React + Vite + TypeScript + React Router
- DB: SQLite (`server/db.sqlite`)

## Dev
Terminal 1:
```bash
cd server
npm run dev
```

Terminal 2:
```bash
cd client
npm run dev
```

API: http://localhost:4000/api

Web: http://localhost:5173
